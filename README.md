# Slime Survival

Projeto inspirado em Vampire Survivors desenvolvido em Unity.

## 🎮 Features
- Sistema de combate automático
- Inimigos com spawn contínuo
- Coleta de XP
- Level up
- Sistema de Upgrades

## 🛠️ Tecnologias
- Unity
- C#
